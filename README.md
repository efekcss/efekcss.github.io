# Teknoloji Stack ve Yapılan İşler

**Frontend & Styling :**
* **HTML5 - Semantik** yapı

* **Google Fonts** - Inter font family
* **Material Symbols** - İkon kütüphanesi
**JavaScript :**
* **Vanilla JavaScript** - Framework'siz, saf JS
* **CSS Animations** - Özel keyframe animasyonlar
**Hosting & Deployment :**
* **GitHub Pages** - Statik site servisi

## Özel Tasarım Elemanları

**Animasyonlar**

```bash
    moveHorizontal - Yatay hareket
    moveInCircle - Dairesel rotasyon
    moveVertical - Dikey hareket
```

Custom Styles
* shell.glass - Glassmorphism efekti (blur + transparency)
* .frost - Frost efekti (primary renk tinted blur)
* .glow-hover - Hover'da ışıltı efekti
Renkler
* Primary: #8c9de8 (Lavender blue)
* Background Light: #f6f6f8
* Background Dark: #000000

## Özellikler
-  Dark mode (default)
-  Fully responsive (mobile-first)
- No build tools (pure static HTML)
- Smooth animations
